## ✨ 주요 정보 수집
> 본 저장소는 필요한 카드 상품 정보, 혜택 데이터, 실적 기준을 수집하는 Python 기반의 크롤링/파싱 모듈입니다. <br>
> 웹상의 비정형 데이터를 수집 및 가공하여 메인 앱 서비스가 사용자에게 정확한 정보를 제공할 수 있도록 지원합니다.

## 🛠️ 가상환경 실행 및 라이브러리 설치

source crawling/bin/activate
